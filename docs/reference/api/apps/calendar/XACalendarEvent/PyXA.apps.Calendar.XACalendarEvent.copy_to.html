XACalendarEvent.copy_to

XACalendarEvent.copy_to(calendar: PyXA.apps.Calendar.XACalendar) → PyXA.apps.Calendar.XACalendarEvent[source]

Makes a copy of this event and places it on the specified calendar.

Parameters

calendar (XACalendar) – The calendar to copy the event to.

Returns

A reference to this event object.

Return type

XACalendarEvent

Example

Copy today’s event to another calendar

>>> import PyXA
>>> app = PyXA.application("Calendar")
>>> calendar = app.default_calendar()
>>> calendar2 = app.calendar(2)
>>> event = calendar.events_today()[0]
>>> event.copy_to(calendar2)
